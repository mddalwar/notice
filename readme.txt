If you want to show Institute Notice in your WordPress website, you can use this. You can be upload your notice file with this plugin options. You can use this without any configure. It's very easy to use.

== Description ==
Any one can use Institute Notice for showing institute notices in their wordpress website. You don't need to make payment any amount for using Institute. So let's enjoy.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/institute-notice` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Dashboard > Notice.
4. You will be find a option to add notices.
5. Anywhere you can be use '[ins_notice]' to show all of the notices.
6. You can be set maximum notice amount in the notice page using 'notice_count' attribute.
7. So, Let's Enjoy.


== Frequently Asked Questions ==

= Need any configuration for this? =

No, You don't need any configure for this plugin.

= How can I change notice limit? =

You will be able to change total notice limit using 'notice_count' attribute.
